# UAV Assistive Operation Application

It is an application developed using Universal Windows Platform (UWP) to improve accessibility when operating UAV systems. The application is designed to support Xbox wireless controllers and the DJI Mavic Air. This project aims to reduce limitations in UAV operation for users with dexterity and mobility difficulties by remapping existing acessible gaming controllers to UAV controls. <br>

---

### Installation

<ol>
  <li> Clone the repository </li>
  <li> Open the solution in Visual Studio</li>
  <li> Install Universal Windows Platform development </li>
  <li> Build the solution </li>
</ol>

### Requirements
- A windows PC or laptop
- An Xbox wireless controller
- A DJI Mavic Air
- A WiFi dongle

---

### Features

- Controller connection management
- Controller input remapping
- Dynamic remapping validation and error feedback

* UAV connection management
* Real-time UAV telemetry monitoring
* Live UAV position tracking
* Comprehensive UAV flight control functionality
* Integrated UAV safety monitoring and protection systems
* UAV simulator testing mode

- Event Logging
- Support for both metric and imperial units
